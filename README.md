# 🏥 Medicine Inventory Tracker

This is a **Full Stack Web Application** to manage your medicine inventory.

---

## ✨ Features

✅ Add new medicines  
✅ View all medicines  
✅ Delete medicines  
✅ Clean and responsive UI  

---

## ⚙️ Technologies Used

| Layer       | Technology            |
|-------------|------------------------|
| Frontend    | Angular                |
| Backend     | Node.js, Express       |
| Database    | MongoDB                |
| Styling     | CSS Flexbox, Custom Fonts |

--
Backend:
cd medicine-inventory-backend
npm install

Frontend:
cd ../medicine-inventory-frontend
npm install


▶️ Start the Applications
Backend:
npm start
Frontend:
ng serve

✅ Frontend URL:
http://localhost:4200

✅ Backend URL:
http://localhost:3000
